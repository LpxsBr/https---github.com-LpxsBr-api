# API WITH NODE JS

## POST - Add User

Route: ``` host/users/add ```

#### Body example for this API

```
{
    "username": "sample user",
    "email": "user@sample.com",
    "password": "password@sample#123"
}
```

## GET - List Users

``` host/ ``` or ``` host/{name}```

return welcome messages

### Get Users

#### Get All Users

``` host/users/list ```

#### Get an user by id

``` host/user/{id} ```
